What this is

This is a tool for cleaning up PDF text. Sometimes, when text is recognised in a scanned PDF text, some technical mistakes appear. Lines are ended with a new paragraph mark even when it is still the same paragraph. Although it is not always possible to automatically find these errors, some cases are easy to fix. For example, if a paragraph mark is folowed by a small letter, that means that a new sentence doesn't start and probably it is still the same paragraph. This simple tool is designed to fix just that.

What this is not

This is not a spellchecker. Nor is it a grammar checker. If a word is written wrong, it wiil not bee corrected :)

How to use it

Enter the input text in the left field. It will be fixed as much as possible and displayed on the right. You can also copy-paste text.

The settings can also be changed. If you need a specific combination of corrections, you can change what is being fixed. By default, most of the features are enabled.
